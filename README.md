# VsHx

---

## Motions

### Control
```

Toggle HX:    ctrl+h
Exit HX:      h
Quit Action:  q

```

### Simple Motions
```

Up:           i
Down:         k
Left:         j
Right:        l

```

### Complex Motions
```

Next Word:    w
Prev Word:    b
Word Start:   e
Select Line:  x

```

### Modifiers
```

Select:        shift+{i j k l w b e}
Snap:          G {i j k l}
Snap-Select:   G shift+{i j k l}
View:          V {i j k l}
View-Select:   V shift+{i j k l}
Area-Select:   A shift+{i j k l}
Repeat:        <num> {i j k l w b e x}
Move Selected: M {i j k l}

```

### Insert Actions
```

Replace:       r
Open Line:     o
Open Line Top: shift+o

```

### Manipulation Actions
```

Delete:       d
Undo:         u
Redo:         shift+u
Copy:         c
Paste:        p
Reg Copy:     shift+c <reg> enter
Reg Paste:    shift+p <reg> enter

```

### Search Actions
```

Jump To:	f <char> enter
Jump Untill:	t <char> enter  // moves to symbol right before match
Select To:	s {f t} <char> enter

```

### Surround Actions
```

Select Inner:  s i {( [ { <}
Select Outer:  s o {( [ { <}
Wrap With:     s w {( [ { <}

```

### Complex Manipulation Actions
```

To Lowercase: space u
To Uppercase: space shift+u
Join Lines:   space j
Split:        space s <char>
Find:	      space f <find>
Find&Select:  space n <find>

```
